## 🚀 Cara Memulai

### 1. Install & Run (2 menit)
```bash
npm install
npm run dev
```

### 2. Customize Data (10 menit)
Edit `lib/constants.ts`:
- Nama masjid
- Alamat & kontak
- Jadwal sholat
- Kegiatan
- Nominal donasi

### 3. Upload Gambar (5 menit)
- Upload ke `public/`
- Update path di komponen

### 4. Test & Deploy (10 menit)
- Test di mobile & desktop
- Build: `npm run build`
- Deploy ke Vercel

**Total Setup Time: ~30 menit**
